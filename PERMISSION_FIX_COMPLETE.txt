🚀 NEXT STEPS

On Your Debian Server:
───────────────────────
1. Configure passwordless sudo:
   ```bash
   echo "ggnet ALL=(ALL) NOPASSWD: ALL" | sudo tee /etc/sudoers.d/ggnet
   sudo chmod 0440 /etc/sudoers.d/ggnet
   ```

2. Install runner as ggnet user:
   ```bash
   cd /opt/actions-runner
   ./config.sh --url https://github.com/... --token ... --user ggnet
   ```

3. Install as service:
   ```bash
   sudo ./svc.sh install
   sudo ./svc.sh start
   ```

4. Verify runner is online in GitHub settings

5. Run deployment workflow:
   - Go to Actions tab
   - Select "Debian Production Deployment"
   - Click "Run workflow"
   - Watch it deploy! 🎉
