📥 INSTALL PYTHON

Option 1: Official Python (Recommended)
  1. Download: https://www.python.org/downloads/
  2. Install Python 3.11+ or 3.12+
  3. ⚠️  IMPORTANT: Check "Add Python to PATH" during installation!
  4. Restart terminal

Option 2: Microsoft Store
  1. Open Microsoft Store
  2. Search "Python 3.11" or "Python 3.12"
  3. Install
  4. Restart terminal

Verify installation:
  python --version
  # Should show: Python 3.11.x or 3.12.x

🚀 MANUAL BACKEND TESTING (Without pytest)

Step 1: Start Backend
---------------------
cd backend
python run.py

# Should see:
# 🚀 Starting ggNet Backend...
# ✅ Database initialized
# ✅ Server running at http://0.0.0.0:5000

Step 2: Test API Endpoints
---------------------------

Using curl (if available):
  curl http://localhost:5000/api/status
  curl http://localhost:5000/api/v1/machines

Using browser:
  http://localhost:5000/docs
  → Interactive API documentation
  → Click "Try it out" on any endpoint

Using PowerShell:
  Invoke-WebRequest -Uri "http://localhost:5000/api/status" | Select-Object -Expand Content
  Invoke-WebRequest -Uri "http://localhost:5000/api/v1/machines" | Select-Object -Expand Content

Using Python (in new terminal):
  python -c "import requests; print(requests.get('http://localhost:5000/api/status').json())"
